================================================================================
JSON FILE STORAGE LAYER
================================================================================

All data lives in these JSON files:

data/jira_issues.json
--------------------------------------------------------------------------------
Array of issue objects. Each object:
{
  "issue_id"           : "PROJ-1001",
  "title"              : string,
  "description"        : string,
  "comments"           : [string],
  "resolution"         : string,
  "resolution_comment" : string,
  "root_cause"         : string,
  "labels"             : [string],
  "component"          : string,
  "priority"           : "Critical|High|Medium|Low",
  "issue_type"         : "Bug|Task|Story|Incident",
  "status"             : "Open|In Progress|Closed",
  "time_to_close"      : float (days),
  "created_at"         : ISO datetime string,
  "closed_at"          : ISO datetime string
}

data/sessions/{session_id}.json
--------------------------------------------------------------------------------
One file per active chat session:
{
  "session_id"  : string,
  "history"     : [
    {
      "user"      : string,
      "bot"       : string,
      "timestamp" : ISO datetime string
    }
  ],
  "context": {
    "last_api"    : string or null,
    "last_error"  : string or null,
    "last_issues" : [string],
    "last_intent" : string or null
  },
  "created_at"  : ISO datetime string,
  "last_active" : ISO datetime string
}

data/patterns/clusters.json
--------------------------------------------------------------------------------
Generated after clustering:
[
  {
    "cluster_id"              : int,
    "size"                    : int,
    "top_keywords"            : [string],
    "avg_resolution_hours"    : float,
    "representative_issues"   : [string]
  }
]

models/jira_meta.json
--------------------------------------------------------------------------------
Metadata array for FAISS index lookup:
[
  {
    "issue_id"           : string,
    "title"              : string,
    "resolution"         : string,
    "resolution_comment" : string,
    "priority"           : string,
    "component"          : string,
    "time_to_close"      : float,
    "labels"             : [string]
  }
]

services/json_store.py
================================================================================

Build class JsonStore with these methods:

__init__(self):
  self.issues_path    = "data/jira_issues.json"
  self.sessions_dir   = "data/sessions"
  self.patterns_path  = "data/patterns/clusters.json"
  self.meta_path      = "models/jira_meta.json"
  os.makedirs("data/sessions", exist_ok=True)
  os.makedirs("data/patterns", exist_ok=True)
  self._issues_cache  = None    <- in-memory cache after first load

load_issues(self) -> list:
  If cache exists return cache
  Read data/jira_issues.json
  Store in self._issues_cache
  Return list

search_issues(self, filters: dict) -> list:
  Load all issues
  Filter by: priority, issue_type, status,
             component, date range, keyword in title/description
  Return filtered list

get_issue_by_id(self, issue_id: str) -> dict or None:
  Load all issues
  Return first match on issue_id field

get_dashboard_stats(self) -> dict:
  Load all issues
  Compute:
    total_issues           = len(issues)
    patterns_discovered    = count from clusters.json
    avg_resolution_hours   = mean of time_to_close * 24
    duplicate_rate         = count(resolution=="Duplicate") / total
    issue_type_distribution= Counter on issue_type field
    priority_distribution  = Counter on priority field
    daily_trend            = group by created_at date, last 30 days
    top_patterns           = top 5 from clusters.json by size
    recent_critical        = last 5 Critical+Open issues
  Return as dict

load_session(self, session_id: str) -> dict:
  path = data/sessions/{session_id}.json
  If file exists: read and return
  Else return new empty session dict

save_session(self, session_id: str, session: dict):
  path = data/sessions/{session_id}.json
  Write session dict as JSON with indent=2

cleanup_sessions(self, expire_minutes=30):
  For each file in data/sessions/
    If last_active older than expire_minutes: delete file

load_patterns(self) -> list:
  Read data/patterns/clusters.json
  Return list (empty list if file not found)

save_patterns(self, clusters: list):
  Write to data/patterns/clusters.json with indent=2

load_meta(self) -> list:
  Read models/jira_meta.json
  Return list

save_meta(self, meta: list):
  Write to models/jira_meta.json with indent=2

conversation_memory.py
================================================================================

Class ConversationMemory using JsonStore:

__init__(self):
  self.store     = JsonStore()
  self.max_turns = 10
  self._cache    = {}     <- active sessions in RAM

get_or_create(self, session_id) -> dict:
  If session_id in self._cache: return it
  Load from json_store.load_session(session_id)
  Store in cache and return

update(self, session_id, user_msg,
       bot_response, entities, intent):
  session = get_or_create(session_id)
  Append to history (keep last max_turns only)
  Update context fields:
    last_api    <- entities["apis"][0] if present
    last_error  <- entities["errors"][0] if present
    last_intent <- intent
    last_active <- datetime.now().isoformat()
  Update cache
  Save via json_store.save_session()

enrich_with_context(self, entities, session_id) -> dict:
  session = get_or_create(session_id)
  ctx = session["context"]
  If entities["apis"] empty and ctx["last_api"] exists:
    entities["apis"] = [ctx["last_api"]]
  If entities["errors"] empty and ctx["last_error"] exists:
    entities["errors"] = [ctx["last_error"]]
  Return enriched entities

STARTUP SEQUENCE (main.py)

@app.on_event("startup")
async def startup():
  Step 1: Create all directories
    data/, data/sessions/, data/patterns/, models/

  Step 2: Validate jira_issues.json exists
    If not found: print warning + create empty file

  Step 3: Load ML models via ModelService singleton
    Load Word2Vec, BiLSTM, tokenizer,
    label encoder, regressor

  Step 4: Load FAISS index
    Load jira.index + jira_meta.json
    If not found: run build_index on jira_issues.json

  Step 5: Start session cleanup background task
    Every 10 minutes: cleanup_sessions(expire_minutes=30)

  Step 6: Print startup summary
    Total issues loaded: N
    FAISS index size: N
    Models ready: Yes/No

ML PIPELINE
================================================================================

ml/train_embeddings.py:
--------------------------------------------------------------------------------
  Load data/jira_issues.json
  Merge title + description + comments + resolution_comment
  Clean text (preprocessor.py)
  Train Word2Vec:
    vector_size=128, window=5,
    min_count=2, sg=1, epochs=30
  Save to models/jira_word2vec.model
  Print: vocab size, top 20 words

ml/train_model.py:
--------------------------------------------------------------------------------
  Load jira_issues.json
  Build labels from resolution + issue_type + priority
  Build tokenizer -> save models/tokenizer.pkl
  Build label encoder -> save models/label_encoder.pkl
  Build embedding matrix from Word2Vec
  Build BiLSTM + Attention model:
    Embedding(pretrained) ->
    BiLSTM(128, return_sequences=True) ->
    BiLSTM(64, return_sequences=True) ->
    AttentionLayer(64) ->
    Dense(128, relu) -> Dropout(0.4) ->
    Dense(64, relu) ->
    Dense(num_classes, softmax)
  Train Phase 1 (frozen embeddings, 50 epochs, lr=1e-3)
  Train Phase 2 (unfrozen, 10 epochs, lr=1e-4)
  EarlyStopping, ReduceLROnPlateau, ModelCheckpoint
  Save best model to models/best_model.keras
  Print: accuracy, per-class F1 score

ml/build_index.py:
--------------------------------------------------------------------------------
  Load jira_issues.json
  Load Word2Vec model
  For each issue:
    text = clean(title + description + resolution_comment)
    vector = mean of word vectors -> L2 normalize
  Build FAISS IndexFlatIP(128)
  Add all vectors
  Save index: faiss.write_index -> models/jira.index
  Save metadata: models/jira_meta.json via JsonStore
  Print: total indexed, index size

services/model_service.py (Singleton):
--------------------------------------------------------------------------------
  Load on first instantiation:
    * Keras BiLSTM model
    * Tokenizer (pickle)
    * LabelEncoder (pickle)
    * Resolution time regressor (sklearn, pickle)
  predict_pattern(text)       -> label, confidence, top3
  predict_intent(text)        -> intent string
  estimate_resolution(feats)  -> float hours

services/vector_store.py:
--------------------------------------------------------------------------------
  FAISS IndexFlatIP + metadata list
  get_vector(text): Word2Vec mean -> L2 normalize
  add_issues(issues, preprocessor): bulk index
  search(query, preprocessor, top_k): return ranked list
  save(index_path, meta_path)
  load(index_path, meta_path)
